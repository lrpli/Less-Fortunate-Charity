## 🚀 Getting Started

### View the Website
1. **Open in Browser:**
   ```
   Open index.html directly in your web browser
   ```

2. **Or use a Local Server:**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Or Node.js
   npx http-server
   
   # Then visit: http://localhost:8000
   ```
